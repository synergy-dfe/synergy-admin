# Synergy.Admin.Web

Portal administrativo da plataforma **Synergy**.

## Stack

- React
- Vite
- JavaScript

## Scripts

```bash
npm install    # Instalar dependências
npm run dev    # Servidor de desenvolvimento
npm run build  # Build para produção
npm run preview # Preview do build
```

## Login (temporário)

- **Usuário:** admin
- **Senha:** admin

## Estrutura

- **Empresa** → **Tenant** → **Apps liberados**
- Cadastro de empresas, tenants e aplicações
- Liberação de apps por tenant
- Arquitetura preparada para hospedar apps externos embedded

## Desenvolvimento

Abra o terminal na pasta `Synergy.Admin.Web` e execute `npm run dev`.
